## 一、申报版本与功能边界

本次申请版本对应代码仓库主干 V2.0 状态，包含以下 6 大核心模块（前一版本 V1.0 仅含其中 1-3 项的最小实现）：

1. **跨文化对话**：以选定市场（AFRICA / EU / US / SEA / LATAM / DEFAULT）的文化规则注入系统提示词的多轮对话；
2. **视频脚本生产流水线**：DeepSeek 文案 + SiliconFlow Wan2.2-T2V 视频生成的异步任务编排；
3. **文化素材库**：抓取式、文本式两条录入路径，含我的素材、自定义标签、CSV/JSON 导出；
4. **文化适配评分**：DeepSeek `json_object` 模式按「语气贴合 / 文化禁忌 / 本地化 / 可信度 / 受众共鸣」5 维度评分，雷达图可视化；
5. **品牌音调规则包**：把品牌关键词、禁用词、风格说明保存为可挂载的「Brand Voice」，对话与任务可一键复用；
6. **反馈闭环**：AI 回复 / 评分结果均可一键「有用 / 无用」，管理员侧汇总按市场满意度。
